# 3anGL - DSL Editor

**3anGL** is a web-based Domain Specific Language (DSL) editor built with **React**, **TypeScript**, and **Monaco Editor**. It allows editing a custom orchestration language with real-time syntax highlighting for fields, subfields, and instructions.

---

## Features

- **Fields Highlighting**:  
  Fields section (`% Fields`) and subfields (like `<subfield>`) are highlighted with green shades.
- **Instructions Highlighting**:  
  Instruction lines starting with `*` (`% Instructions`) are highlighted in blue.
- **Real-time Decoration**:  
  Highlighting updates immediately as the user types.
- **React + Monaco Editor Integration**:  
  Smooth editor experience with line numbers, word wrap, and custom theming.
- **TailwindCSS Styling**:  
  Modern, responsive design.
